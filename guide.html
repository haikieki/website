いいリフォームをするために

1. リフォーム前のチェックポイント

リフォームを成功させるためには、事前の準備が欠かせません。 まずは現在の住まいの状況を正しく把握し、何を改善したいのかを明確にしましょう。

✓

目的を明確にする

「なぜリフォームするのか」を家族で話し合い、優先順位をつけましょう。見た目の改善、機能向上、老朽化対策など、目的によって最適な方法が異なります。

✓

予算を決める

リフォームにかけられる予算の上限を決めておきましょう。予備費として総額の10〜15%を確保しておくと安心です。

✓

情報収集をする

リフォーム事例、費用相場、使える補助金制度などを調べておきましょう。知識があることで業者との打ち合わせもスムーズになります。

2. 信頼できる業者の選び方

業者選びはリフォーム成功の最も重要なポイントです。 価格だけで判断せず、総合的に信頼できる業者を選びましょう。

✓

資格・実績を確認する

全国住宅リフォーム取扱主任者などの資格を保有しているか確認しましょう。資格は専門知識と倫理観の証です。

✓

複数の業者から見積もりを取る

最低でも2〜3社から見積もりを取り、内容を比較しましょう。極端に安い見積もりには注意が必要です。

✓

契約内容を確認する

工事内容、金額、工期、保証内容などが明記された契約書を必ず確認しましょう。口約束だけで工事を始めないことが大切です。

3. リフォーム中の注意点

工事が始まってからも、確認すべきポイントがあります。 定期的に進捗を確認し、気になる点は早めに相談しましょう。

✓

工事の進捗を確認する

定期的に現場を確認し、予定通り進んでいるか確認しましょう。写真を撮っておくと後で役立ちます。

✓

変更はすぐに書面で確認する

工事中に仕様変更が生じた場合は、必ず書面で内容と追加費用を確認しましょう。

✓

近隣への配慮

工事前に近隣へ挨拶をしておきましょう。騒音や工事車両の問題を事前に伝えておくことでトラブルを防げます。

4. トラブル時の対応

万が一トラブルが発生した場合は、冷静に対処することが重要です。 記録を残し、適切な相談窓口を活用しましょう。

✓

記録を残す

問題が発生した箇所の写真を撮り、やり取りの記録を残しましょう。日時と内容をメモしておくことが大切です。

✓

まず業者に相談する

感情的にならず、具体的な問題点を伝えて改善を求めましょう。書面でのやり取りが望ましいです。

✓

公的機関に相談する

解決が難しい場合は、消費生活センター（局番なし188）や住宅リフォーム・紛争処理支援センターに相談しましょう。
